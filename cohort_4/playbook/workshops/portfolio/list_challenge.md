- Let's talk about lists. Lists can be created with either the `ul`
    (unordered) or `ol` (ordered) list tags. We put `li` tags inside of `ul`
    or `ol` for each element.

    ```html
    <h1>Unordered List</h1>
    <ul>
      <li>Eggs</li>
      <li>Lime</li>
      <li>Sharp cheddar</li>
    </ul>
    <h1>Ordered List</h1>
    <ol>
      <li>October</li>
      <li>November</li>
      <li>December</li>
    </ol>
    ```
